# Логотипы команд РПЛ

Положи сюда файлы с эмблемами клубов, и они автоматически появятся в матч-центре
мини-аппа (карточка матча наверху). Если файла нет — приложение просто покажет
кружок с инициалами команды, ничего не сломается.

## Формат

- PNG, прозрачный фон
- квадрат 256x256 px (можно чуть больше/меньше — впишется в круг, но не сильно
  прямоугольный, иначе обрежется по кругу)
- по возможности до ~60 КБ на файл — не критично, но так приложение будет быстрее

Если в следующем сезоне в лигу придёт новая команда, для неё нужно будет:
1. добавить сюда файл с логотипом по тому же принципу;
2. добавить одну строку в `TEAM_LOGO_SLUGS` в `src/App.tsx` (сопоставление
   кириллического названия команды и имени файла).
