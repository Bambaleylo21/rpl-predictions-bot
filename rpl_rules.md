# Правила турнира прогнозов РПЛ

## 1. Формат сезона

Сезон РПЛ разбит на два этапа:
— Осенний этап — туры 1 – последний в году (в зависимости от последнего тура декабря)
— Весенний этап — (первый тур нового года) – 30.

Внутри каждого этапа участники разбиты на две лиги: Высшую и Низшую. Таблица и начисление очков считаются отдельно в рамках своей лиги.

По итогам Осеннего этапа происходит переход между лигами: 2 лучших участника Низшей лиги повышаются в Высшую, 2 худших участника Высшей лиги понижаются в Низшую, остальные остаются в своей лиге. После этого автоматически открывается Весенний этап с пересчитанным составом лиг. Полностью новый сезон открывается только после завершения Весеннего этапа.

## 2. Как начисляются очки за прогноз матча

— 4 очка — угадан точный счёт
— 2 очка — угадан исход и точная разница мячей
— 1 очко — угадан только исход, без учёта разницы
— 0 очков — исход не угадан

Каждый тур и каждый матч оцениваются одинаково.

## 3. Правила самого прогноза

Прогноз — счёт матча. Его можно ставить и менять сколько угодно раз, пока матч не начался. С момента стартового свистка (по плановому времени) прогноз блокируется навсегда. Если прогноз не поставлен до начала матча — очков за него просто нет.

## 4. Турнирная таблица — порядок сортировки при равенстве очков

1. общая сумма очков (с бонусами) — по убыванию
2. количество точных счетов — по убыванию
3. количество угаданных «исход + разница» — по убыванию
4. количество угаданных «только исход» — по убыванию
5. процент угадывания — по убыванию
6. количество сделанных прогнозов — по возрастанию
7. дата последнего прогноза — раньше сделанный даёт преимущество

## 5. «Оценка ИИ»

Вкладка «Оценка ИИ» в Матч-центре показывает процентную оценку исходов матча (победа хозяев / ничья / победа гостей), рассчитанную собственной статистической моделью приложения — упрощённой Пуассон-моделью на истории голов команд, с учётом прошлых сезонов и текущей формы.

Это не букмекерские коэффициенты и не официальный прогноз, а справочная информация на основе статистики, без гарантии точности. Настоящих коэффициентов на матчи РПЛ в используемых данных нет.

## 6. Взносы

Сумма взноса на участие определяется перед стартом каждого этапа путём голосования в телеграм-канале — отдельно участниками каждой лиги. Сначала определяется взнос Низшей лиги, затем — Высшей, поскольку взнос в Высшей лиге не может быть меньше, чем в Низшей.

## 7. Победители

Победителем лиги становится участник, занявший 1-е место в своей лиге, — он получает 100% от призового фонда. Победителем не может стать ИИ: в этом случае первое место присуждается ближайшему человеку.
